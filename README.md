# ssmwrap

ssmwrap execute commands with environment variables loaded from AWS SSM Parameter Store.

## Usage

```console
$ ssmwrap -paths /production -- your_command
```

## Install

```console
$ go get github.com/handlename/ssmwrap/cmd/ssmwrap
```
